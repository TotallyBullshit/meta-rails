MetaQuerier

What is MetaQuerier?

MetaQuerier is a RubyOnRails plugin that is a part of MetaRails. Its aim is to make any kind of query to a database in a RubyOnRails application with zero configuration.

MetaQuerier detects the structure of the application models. Shows a diagram of the application and offers a custom form for building queries and browsing its data.

Install

Prerequisites: In order to show graphs of the application structure, MetaQuerier needs Graphviz.

  1. From the rails application root directory: 
    script/plugin install svn://rubyforge.org/var/svn/meta-rails/HEAD/meta_querier
  2. Reload your server and point your browser to the "/mega_querier" folder of your application, for example: 
    http://localhost:3000/meta_querier

MetaQuerier looks if you have ActAsAuthenticated plugin installed, if so it calls automatically the function login_required acting as expected.
If you have ActAsAuthenticated plugin installed but you don't want MetaQuerier to use ActAsAuthenticated you must add the next code to config/environment.rb: 

MetaQuerierUseActAsAuth = false

Caution!

MetaQuerier is in alpha version and still haves security issues be careful and not use it in production environments.

We are fixing a bug that can be a backdoor for code injection.

Future Features

  • My Queries: Manage common queries in order to access them fast.
  • Database Regions: Divide the database in multiple regions for better browsing of large databases.
  • Export query data to CSV and XML
  • Generate the queries clicking directly in the database diagram instead of using provided forms.
