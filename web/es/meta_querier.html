MetaQuerier

¿Qué es MetaQuerier?

MetaQuerier es un plugin para RubyOnRails que forma parte de MetaRails. Su objetivo es hacer cualquier tipo de consulta a un base de datos de una aplicación RubyOnRails sin ningún tipo de configuración.

MetaQuerier detecta la estructura de los modelos de la aplicación. Muestra un diagrama de la aplicación y ofrece un formulario para generar consultas y navegar por sus datos.

Instalar

Prerrequisitos: Para mostrar gráficos de la estructura de la aplicación, MetaQuerier necesita Graphviz.

  1. Desde el directorio raíz de la aplicación rails: 
    script/plugin install svn://rubyforge.org/var/svn/meta-rails/HEAD/meta_querier
  2. Recargar el servidor y acceder con el navegador al subdirectorio "/mega_querier" de la aplicación, por ejemplo: 
    http://localhost:3000/meta_querier

MetaQuerier mira si tienes instalado el plugin ActAsAuthenticated, en caso afirmativo llama automáticamente a su método login_required.
Si a pesar de tener el plugin ActAsAuthenticated plugin instalado no deseas que MetaQuerier lo use, tienes que añadir la siguiente línea a config/environment.rb: 

MetaQuerierUseActAsAuth = false

Precaución!

MetaQuerier esta en versión alpha y todavía tiene problemas de seguridad no lo uses en entornos de producción.

Estamos arreglando errores que pueden ser utilizados para realizar ataques de injección de código.

Futuras Características

  • My Queries: Gestión de consultas comunes para acceder a ellas rápidamente.
  • Database Regions: Dividir la base de datos en múltiples regiones para poder explorar mejor bases de datos grandes.
  • Exportar las consultas a CSV y XML.
  • Generar consultas clicando directamente en el diagrama de la base de datos en lugar de los formularios.
