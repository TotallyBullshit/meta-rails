new(options = {})

Creates a object for generate models diagram.

Takes a options hash parameter that must have this value:

  • :model_names => Array of model names (later .constantize will be applied to them)
  • :model_columns => Hash with keys equal to model names (in .classify format), whose value is an array with the column names.
  • :model_associations => The same as :model_columns but in each position the value is the associated models (in .classify format).
  • :actual_model [OPTIONAL] => this model will be highlighted.

Uses mq_diagram.dot.erb template in order to create de .dot file that Graphviz will take for generate the diagram.
